# youdo

## Usage

### Adding a new task to a file:

```
youdo add -t "Wynieść śmieci" -p ./house.youdo
```

### Listing tasks from a given file:

```
youdo list -p ./house.youdo
```

### Syntax:

youdo action (OPTIONAL -t task) -p path
